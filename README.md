# Physical Optics Key Points

This repository contains review materials and key points for Physical Optics.

## Contents

The repository includes documents covering the following topics:
- **Polarization of Light** (`光的偏振.docx`)
- **Diffraction of Light** (`光的衍射.docx`)
- **Double Slit Interference** (`双缝干涉.docx`)
- **Thin Film Interference** (`薄膜干涉.docx`)

These documents are intended to help with understanding the core concepts of physical optics.
